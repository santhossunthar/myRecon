# myRecon
Automated Recon tool to gather information of target for bugbounty hunters with database support.

## Getting Started

### Installation
```sh
  git clone https://github.com/santhossunthar/myRecon.git
  cd myRecon
  chmod +x *.sh
  ./recon.sh test.com test_db
 ```
